To use the JSON Compiler Service send a POST request containing a schema in the LAPPS Alternative syntaX (LAX) format. See the LAX GitHub respository for details on the syntax.

Headers

Please specify the following headers in the POST request:

Header Value
Content-Type text/plain
Accept application/json

Example

Suppose the file schema.txt contains the following

            
 properties {
     name {
         type string
     }
     age {
         type integer
         minimum 0
     }
 }

curl -X POST -H "Accept: application/json" -H "Content-type: text/plain" -d @schema.txt http://grid.anc.org:9080/json-compiler
